# 小龚PDF阅读器 (XiaoGongPDF)

一款基于 MuPDF 开发的轻量级 PDF 阅读器，专为中国用户打造。

## 核心特点

✅ **基于 MuPDF 引擎** - 采用开源 MuPDF 库，快速、稳定、高质量渲染
✅ **完全本地化** - 针对中国用户习惯优化，界面简洁易用
✅ **零网络代码** - 不含任何网络通信功能
✅ **隐私保护** - 绝不后台上传任何数据，100% 本地运行
✅ **开源透明** - 源代码完全公开，欢迎审查

## 主要功能

- 📄 快速打开和浏览 PDF 文档
- 🔍 缩放、旋转、适应页面
- 📑 缩略图导航
- ⌨️ 丰富的快捷键支持
- 🖥️ 全屏模式
- 📊 多标签页浏览

## 系统要求

- Windows 7 或更高版本
- Visual Studio 2019 或更高版本（编译需要）
- MuPDF 库

## 编译说明

1. 克隆本仓库
2. 使用 Visual Studio 打开 `XiaoGongPDF.sln`
3. 配置 MuPDF 库路径
4. 编译运行
